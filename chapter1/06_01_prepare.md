# 基础切割包

通常情况下，我们会建立一个基础的切割包，每次项目都基于此进行丰富内容。

## 目录结构

```
文件夹/
    ├── css/
    ├── js/
    └── image/
    └── pic/
    └── index.html
```

* 文件夹的根目录中存放切割的 HTML 文件；

* css 文件夹中存放页面切割中使用到的样式表文件；

* js 文件夹中存放页面切割中使用到的 js 文件，jQuery 库等；

* image 文件夹中存放页面切割中使用的修饰性图片，如：背景图片、ICON图标等；

* pic 文件夹中存放页面切割中使用的内容性图片，如：广告图片，轮播图片等；
